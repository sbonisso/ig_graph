Dependencies

IgGraph has been tested on many different versions of Ubuntu, and should work on most unix-based OS. If building from source, the dependencies listed are contained as submodules in the git repository. Instructions for compilation are located below.

  • gcc with C++11 support (GCC 4.7 and later)
  • SeqAn 2.0+
  • TCLAP
  • cereal
  • cpptest
